# KIND Token Experience

A modern Next.js platform presenting the KIND Token mission with a compassionate, data-rich interface for investors, partners, and community members.

## ✨ Features
- Emotionally resonant landing page with animated impact flywheel and real-time investor metrics
- Token transparency center with interactive tokenomics, live analytics, and transaction feed
- Wallet-ready dashboard using Wagmi connectors for MetaMask and WalletConnect
- Interactive impact atlas with project storytelling and funding submission form
- Educational hub, community engagement tools, and compliance-ready contact center
- Tailwind CSS design system capturing KIND’s green-and-white aesthetic with soft gradients and rounded corners

## 🚀 Getting Started

### 1. Install dependencies
```bash
npm install
```

### 2. Run the development server
```bash
npm run dev
```

Visit `http://localhost:3000` to explore the KIND Token experience. Update the `NEXT_PUBLIC_WALLETCONNECT_ID` environment variable for production-ready WalletConnect support.

### 3. Build for production
```bash
npm run build
npm start
```

## 🧰 Tech Stack
- [Next.js 14](https://nextjs.org/) with the App Router
- [React 18](https://react.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Wagmi](https://wagmi.sh/) + WalletConnect for wallet integration
- [Recharts](https://recharts.org/) for interactive data visualization
- [React Simple Maps](https://www.react-simple-maps.io/) for geospatial impact storytelling

## 🔐 Security & Trust
- HTTPS-first deployment (Vercel or AWS recommended)
- Smart contract verification badges and audit references on the Token page
- SOC2-aligned handling for forms and subscriber data (connect to a secure backend service)
